# PigGame
A simple game for two players whose goal is to score 100 points first.

The algorithm is presented as a diagram below
![image](https://user-images.githubusercontent.com/114873018/222556814-5571969a-b14e-4e4a-a4f7-69f0a0cb7c8c.png)

# Stages of the game: 
 - Start of the game. Both players have 0 points.
 ![image](https://user-images.githubusercontent.com/114873018/222557448-a96f30fa-335c-4d5c-95cf-73ac342ec72d.png)
- The player can roll the dice increasing the number of points until he rolls 1. 
  In the case of 1, his points are reset to zero, and the turn passes to the second player.
  Or the player can take the number of points earned.
  ![image](https://user-images.githubusercontent.com/114873018/222558418-71f0ef99-14d0-4bea-9c3f-5846196d5584.png)
  ![image](https://user-images.githubusercontent.com/114873018/222558509-5343167b-2eef-4e5b-a0a9-b949e81458b9.png)
- Victory. Player scores 100 points
![image](https://user-images.githubusercontent.com/114873018/222559566-0f518e94-6448-4c33-a661-4b0fd2efede4.png)
